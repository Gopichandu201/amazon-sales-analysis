# Amazon Product Sales Analysis

This project focuses on analyzing Amazon product sales data to understand product performance, customer ratings, reviews, and discount trends. The objective of this project is to perform end-to-end data analysis and build a dashboard that provides meaningful business insights.

The dataset was first cleaned and processed using Python with the Pandas library. During the data preparation stage, missing values and duplicate records were handled, and the dataset was structured properly for analysis. After cleaning the data, it was inserted into a MySQL database using SQLAlchemy to enable structured querying and data management.

SQL queries were then used to analyze the dataset and generate insights such as the most reviewed products, highest rated products, and category-wise product distribution. These queries helped identify patterns in product ratings, customer engagement, and discount strategies.

To visualize the results, an interactive Power BI dashboard was created. The dashboard includes key performance indicators such as total products, average product rating, total reviews, and total purchases. Additional visualizations were built to show category-wise sales, rating distribution, and the relationship between product discounts and purchase behavior.

This project demonstrates a complete data analytics workflow including data cleaning, database integration, SQL analysis, and business intelligence dashboard creation. The final project was organized and uploaded to GitHub as a portfolio project to showcase practical data analytics skills and the ability to transform raw data into actionable insights.

  

## Overview

This project analyzes Amazon product sales data to uncover insights about product performance, customer ratings, discounts, and purchase trends. The goal is to perform end-to-end data analysis using Python, SQL, and Power BI and build an interactive dashboard to visualize key business insights.

## Tools & Technologies

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* MySQL
* Power BI
* Git & GitHub

## Project Workflow

1. Data Collection – Used an Amazon product dataset containing product details, ratings, reviews, prices, and purchase information.
2. Data Cleaning – Cleaned and prepared the dataset using Python Pandas.
3. Database Integration – Inserted the cleaned dataset into MySQL using SQLAlchemy.
4. SQL Analysis – Performed SQL queries to analyze product categories, ratings, reviews, and discounts.
5. Data Visualization – Built an interactive Power BI dashboard using DAX measures.
6. Portfolio Deployment – Uploaded the project to GitHub for portfolio presentation.

## Key Insights

* Identified top-performing product categories based on purchases.
* Analyzed the relationship between discounts and product sales.
* Found the most reviewed and highest-rated products.
* Evaluated category-wise product performance and rating distribution.

## Dashboard Features

* Category Sales Analysis
* Top Reviewed Products Visualization
* Rating Distribution
* Discount vs Purchase Analysis
